# TESTING.md — Руководство по тестированию

**Проект:** Telegram Mini App «Лазер Тайм»
**Стек:** Vanilla JS, CSS (Telegram theme), localStorage
**Точка входа:** `tg-app/index.html`
**Деплой:** Vercel (автодеплой при push в master)
**Репозиторий:** `lesymos-stack/laser-time-app`

---

## 1. Как запустить

### В браузере (без Telegram)
Просто откройте `tg-app/index.html` в браузере. Приложение работает без сервера.
В браузерном режиме вместо Telegram-кнопок (MainButton, BackButton) появляются fallback-кнопки:
- Стрелка `←` вверху слева — назад
- Кнопка внизу экрана — действие (ЗАПИСАТЬСЯ / ПОДТВЕРДИТЬ / ЗАКРЫТЬ)

### В Telegram
1. Откройте бота [@lasertime_prilo_bot](https://t.me/lasertime_prilo_bot)
2. Нажмите кнопку меню — откроется Mini App
3. Или используйте прямую ссылку на Vercel-деплой

### Сброс состояния
Для чистого тестирования очистите localStorage:
```js
// В консоли браузера (F12):
localStorage.clear();
location.reload();
```

Отдельные ключи:
| Ключ | Что хранит | Формат |
|------|-----------|--------|
| `onboardingDone` | Флаг показа онбординга | `"1"` |
| `offerShown` | Флаг показа модалки скидки | `"1"` |
| `bookingHistory` | Массив записей | JSON-массив объектов |
| `bonusBalance` | Баланс бонусов | Число как строка (`"60.00"`) |

## 9. Известные ограничения

1. **Нет бэкенда** — все данные в localStorage, при очистке теряются
2. **BOOKED_SLOTS** — массив пуст при загрузке, забронированные слоты живут только в рамках сессии (до перезагрузки)
3. **Фото услуг** — массив `photos` пуст у всех услуг, показываются эмодзи категорий
4. **Категории «Коррекция фигуры» и «Лицо»** — содержат placeholder-услуги (педикюр, SPA), не соответствуют названиям категорий
5. **Haptic feedback** — работает только в Telegram, в браузере молча пропускается
6. **state.masterUnlocked** — сбрасывается при перезагрузке (не сохраняется в localStorage)
7. **Расписание** — генерируется от текущей даты, через 14 дней слоты закончатся

## 10. Чек-лист регрессии

Прогоняйте после каждого изменения:

- [ ] Онбординг → «Начать» → главный экран (онбординг полностью исчезает)
- [ ] Модалка скидки → «Пропустить» → не появляется повторно
- [ ] Категория → каталог → услуга → запись → дата → время → подтверждение → успех
- [ ] На экране успеха: запись появилась в «Мои записи»
